# Avenseal Design System

## Tokens

- Background: `#FFFFFF`
- Navy text and anchor surfaces: `#102A43`
- Supporting text: `#334E68`
- Borders and quiet dividers: `#D9E2EC`
- Primary action and positive state: `#2BB673`
- Soft band: `#F5F8FB`

## Typography

Use Geist through `next/font`. Buttons, inputs, tables, sidebars, and status labels all receive explicit font sizing and weights.

## Components

- `Brand`: temporary abstract A/document-fold mark with text wordmark.
- `Button` and `ButtonLink`: emerald, navy, secondary, and ghost variants.
- `PublicShell`: header/footer with approved public navigation only.
- `AdminShell`: navy sidebar and operational content area.
- `StatusBadge`: text label plus visual marker for accessibility.

## Implementation Notes

The generated concepts in `docs/design/` are references. The written visual specification and approval notes override any generated image text drift.
