# Definition of Done

A feature is complete only when applicable requirements are met:

- Product requirements and acceptance criteria are documented and satisfied.
- Tenant isolation and security are reviewed; secrets are not logged.
- Migration safety, rollback or failure behavior, and provider failure isolation are understood.
- Unit, integration, E2E, build, and staging validation pass where applicable.
- Staging fixtures are cleaned up and production remains untouched.
- Documentation, release notes, ADRs, and technical debt are updated.
- The PR is focused, reviewable, CI-passing, and clear about remaining limitations.
